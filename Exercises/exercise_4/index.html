HTML Tag

* An HTML tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document.

* As part of an HTML element, HTML tags help web browsers convert HTML documents into web pages.


What is tag and types?

* The HTML tags can also be simply divided based on basic categories like Basic HTML Root Tags, Formatting tags, Audio and Video Tags, Form and Input Tags, Frame Tags, Link Tags, List Tags, Table Tags, Style Tags, Meta Tags, etc.
